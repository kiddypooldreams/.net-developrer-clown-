Для создания формы в Ruby on Rails вы можете использовать встроенный в Rails помощник form_for. Ниже приведен пример того, как вы можете создать аналогичную форму для создания товара в приложении на Ruby on Rails.

Сначала вам нужно создать новую форму в файле представления (например, new.html.erb), который будет отображаться для создания товара.

  1. Создайте файл app/views/products/new.html.erb:
erb
<h2>Форма создания товара</h2>

<%= form_for @product do |f| %>
  <div>
    <%= f.label :product_name, 'Название товара:' %>
    <%= f.text_field :product_name, required: true %>
  </div>
  <div>
    <%= f.label :quantity, 'Количество:' %>
    <%= f.number_field :quantity, min: 1, required: true %>
  </div>
  <div>
    <%= f.label :price, 'Цена:' %>
    <%= f.number_field :price, step: '0.01', min: 0.01, required: true %>
  </div>
  <div>
    <%= f.submit 'Создать товар' %>
  </div>
<% end %>

Этот код создает форму для объекта @product (предполагается, что у вас есть модель Product), в которой есть поля product_name, quantity и price.

  1. Затем вам нужно добавить соответствующий маршрут и контроллер для обработки создания товара.

В файле config/routes.rb добавьте маршрут для создания товара:

ruby
Rails.application.routes.draw do
  # Добавьте маршрут для создания товара
  resources :products
end

Создайте контроллер ProductsController:

ruby
class ProductsController < ApplicationController
  def new
    @product = Product.new
  end

  def create
    @product = Product.new(product_params)
    if @product.save
      redirect_to @product, notice: 'Товар успешно создан.'
    else
      render :new
    end
  end

  private

  def product_params
    params.require(:product).permit(:product_name, :quantity, :price)
  end
end

Этот контроллер имеет два действия: new для отображения формы создания товара и create для обработки отправленной формы и сохранения товара в базу данных.

  1. Создайте модель Product:
ruby
class Product < ApplicationRecord
  validates :product_name, presence: true
  validates :quantity, numericality: { only_integer: true, greater_than_or_equal_to: 1 }
  validates :price, numericality: { greater_than_or_equal_to: 0.01 }
end

В модели Product вы добавляете валидации для полей товара (название, количество, цена).

Это пример того, как можно реализовать форму для создания товара в приложении на Ruby on Rails. Пожалуйста, убедитесь, что код адаптирован под вашу конкретную структуру проекта и требования вашего приложения.

Если элемент body растягивается за пределы формы, это может происходить из-за стилей, которые могут влиять на размер или ширину элементов. Для того чтобы предотвратить растягивание body, убедитесь, что стили применяются только к элементам формы, а не к body.

Одним из способов исправить это - убедиться, что стили применяются только к элементам внутри формы. Вы можете обернуть стилизованную часть формы в дополнительный контейнер, чтобы ограничить применение стилей только к этой части. Вот пример, как это можно сделать:

html
<%= stylesheet_link_tag 'products', media: 'all', 'data-turbolinks-track': 'reload' %>

<h2>Форма создания товара</h2>

<div class="form-container">
  <%= form_for @product do |f| %>
    <!-- Остальная часть вашей формы здесь -->
  <% end %>
</div>

Затем обновите CSS для формы, чтобы ограничить стили только к этому контейнеру .form-container:

css
/* products.css */

/* Стили для контейнера формы */
.form-container {
  max-width: 400px;
  margin: 0 auto;
}

/* Стили для формы */
.form-container form {
  margin: 20px 0;
  padding: 20px;
  border: 1px solid #ccc;
  border-radius: 5px;
  background-color: #f9f9f9;
}

/* Остальные стили вашей формы */
/* ... */

Это поможет ограничить применение стилей только к части формы, помещенной внутрь контейнера .form-container, и предотвратить растягивание элемента body за пределы формы.

Для добавления метки к тегу <p> в представлении Ruby on Rails, вы можете использовать метод label_tag и метод concat для объединения HTML кода с вашими метками. Например:

erb
<%= label_tag 'custom_label', 'Моя метка:' %>
<p>
  <%= concat label_tag 'custom_label', 'Моя метка:' %>
  Тут ваше содержимое
</p>

Это создаст метку, связанную с тегом <p>. Однако, имейте в виду, что тег <p> не является формой ввода, поэтому обычно не используется с метками. Метки (<label>) обычно используются с полями ввода, такими как <input>, <select> или <textarea>, для улучшения доступности формы и удобства пользователей.

Чтобы подвинуть кнопку чуть вниз и выровнять её по центру внутри формы, можно использовать свойство margin для кнопки в CSS. Для выравнивания элемента по центру можно также использовать свойство text-align: center для обертки формы.

Пример CSS с обновленными стилями для кнопки:

css
/* Общие стили для формы */
form {
  max-width: 400px;
  margin: 0 auto;
  font-family: Arial, sans-serif;
  text-align: center; /* Добавляем центрирование для содержимого формы */
}

/* Стили для группы элементов формы */
.form-group {
  margin-bottom: 15px;
}

/* Стили для меток */
label {
  display: block;
  margin-bottom: 5px;
  font-weight: bold;
}

/* Стили для текстовых полей и полей с числовым вводом */
input[type="text"],
input[type="number"] {
  width: 100%;
  padding: 8px;
  border-radius: 4px;
  border: 1px solid #ccc;
}

/* Стили для кнопки отправки */
input[type="submit"] {
  background-color: #4CAF50;
  color: white;
  padding: 10px 20px;
  border: none;
  border-radius: 4px;
  cursor: pointer;
  font-size: 16px;
  margin-top: 10px; /* Подвигаем кнопку чуть вниз */
}

Это изменение добавляет свойство margin-top: 10px; для кнопки отправки, чтобы подвинуть её немного вниз внутри формы. Также добавлено text-align: center для формы, чтобы выровнять её содержимое по центру. Вы можете настроить значение margin-top в соответствии с вашими предпочтениями отступа.
